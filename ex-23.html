Main Content & The Footer

The header/navigation section of the webpage is not the only section of a webpage.
There is a section where the main information of the webpage would go.

  • The element used to do this is <main> within the <body> element
  • The content inside of the <main> element encompassses the main topics or ideas that will appear on the webpage
  • There should only be one single <main> element in every webpage

Once the weboage is populated with all the content that we want, the <footer> element is used to finish off the webpage. The informaiton in this element usually is beneath the main topics of the webpage

  • The name of the company/creator
  • Some type of copyright information
  • Additional links to other sections of the webpage or to other sites

This section contains important information but it is not suitable for the <header> element
